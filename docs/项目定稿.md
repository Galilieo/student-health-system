# 面向大学生的智能健康习惯管理系统项目定稿

## 项目名称

面向大学生的智能健康习惯管理系统

## 项目目标

本项目面向大学生群体，提供饮食、饮水、睡眠、运动等健康习惯的日常打卡功能，
并通过历史记录、图表分析和智能建议，帮助用户养成规律生活方式。

## 核心功能

### 用户功能

- 登录注册
- 首页健康概览
- 每日打卡
- 历史记录查看
- 图表统计分析
- 个人中心
- 健康提醒
- 智能建议

### 管理员功能

- 用户信息管理
- 公告管理
- 用户反馈管理

## 主要页面

- 登录页
- 注册页
- 首页
- 每日打卡页
- 历史记录页
- 统计分析页
- 个人中心页
- 管理员后台页

## 技术栈

- Android：Java + XML
- 后端：Spring Boot
- 数据库：MySQL
- AI功能：健康建议生成

## 成员分工

- 1号：安卓前端页面与UI
- 2号：安卓功能与交互逻辑
- 3号：后端接口与业务逻辑
- 4号：数据库与后台管理
- 5号：AI建议、测试文档、需求文档、PPT统筹

## 开发原则

- 所有人在 dev 分支开发
- main 作为稳定版本
- 按目录分工，尽量避免多人同时修改同一文件

## 阶段目标

### 第一阶段
完成项目框架搭建、需求定稿、数据库设计、安卓与后端初始化

### 第二阶段
完成登录注册、打卡、历史记录、统计分析等核心功能

### 第三阶段
完成AI建议、提醒功能、后台管理、测试报告、答辩PPT
