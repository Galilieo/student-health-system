# 乐健（Student Health System）

## 1. 项目概述
“乐健”是面向大学生的智能健康习惯管理系统，围绕饮水、睡眠、运动等日常行为构建“记录-反馈-养成”闭环，帮助用户持续改善生活习惯。

## 2. 技术栈
- Android：Java + XML
- 后端：Spring Boot
- 数据库：MySQL

## 6. 接口对接约定
- 返回结构：`{ code, message, data }`
- 认证方式：`Authorization: Bearer <token>`
- 核心接口：
  - `POST /login`
  - `POST /checkin`
  - `GET /summary/today`
  - `GET /history`

## 8. 运行说明（Android）
1. 使用 Android Studio 打开 `android-app/`
2. 同步 Gradle 依赖
3. 连接模拟器或真机后运行 `app`

说明：模拟器访问本机后端可使用 `http://10.0.2.2:8080`，真机请改为电脑局域网 IP。
